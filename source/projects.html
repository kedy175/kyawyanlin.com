University Projects

Multimodal Brain Image Registration Using Deep Learning

As my final year project, I developed a model to align T1 and T2 MRI scans, combining their unique strengths to create a clearer, more comprehensive view of brain structures and abnormalities. By leveraging deep learning, the project aimed to improve accuracy in aligning these two modalities, enhancing diagnostic insights and supporting advanced neuroimaging research.

Hybrid Recipe Recommender System

As a group project, we designed a hybrid recommender system that suggests recipes based on user preferences, combining collaborative filtering (what similar users like) and content-based filtering (matching ingredients and flavors). This approach personalizes recipe recommendations by blending user tastes with ingredient compatibility, helping users discover new dishes that match their palate and dietary needs.

Path Planning And Navigation Using ROS

This group project focused on developing an autonomous navigation system using the Robot Operating System (ROS) integrated with embedded systems. By implementing traditional path planing algorithms, we enabled robots to navigate efficiently through complex environments while avoiding obstacles. The project utilized ROS for simulation and real-time response, along with embedded systems for sensor integration and on-device processing, making it suitable for applications in mobile robotics and automated delivery.
